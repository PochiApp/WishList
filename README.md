# -Wish List- やりたい・叶えたいことリスト 初リリース日：2024/03/28
フォルダー管理するWish Listです。

1年のやりたいこと、読みたい本や、欲しいもの･･･などを1つ1つ分けて管理できます。

操作性を簡単に、テーマカラーやパスコード機能、画像の設定といった自分好みのカスタマイズを可能にしています。
